# 基础

## 数组与切片

```go
// 数组
// 完整初始化
var arr1 = [3]int{10, 20, 30}

// 简化写法（推荐）
arr2 := [3]int{40, 50, 60}

arr3 := [...]int{1, 2, 3} // 长度由元素数量自动推断为 3

// 未显式赋值的元素自动初始化为零值
arr4 := [5]int{1, 2} // [1, 2, 0, 0, 0]
```

```go
slice := []int{1, 2, 3} // 切片（长度可变）
```
